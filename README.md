 📊 Customer Churn Analysis (Power BI)

 📌 Project Overview

This project focuses on analyzing customer churn data using Power BI. The goal is to identify key factors that influence customer churn and provide actionable insights to improve customer retention.

---

 📁 Dataset

* Source: Telco Customer Churn Dataset
* Contains customer details such as:

  * Gender
  * Contract type
  * Payment method
  * Internet service
  * Tenure
  * Churn status

---

📊 Dashboard Features

The Power BI dashboard includes:

* **Churn Rate Overview** – Percentage of customers who churned
* **Churn Distribution** – Comparison of churned vs non-churned customers
* **Churn by Contract Type** – Monthly vs yearly contract analysis
* **Payment Method Analysis** – Impact of payment methods on churn
* **Gender Distribution** – Churn behavior by gender
* **Customer Tenure Analysis** – Relationship between tenure and churn
* **Internet Service Analysis** – Churn across service types

---

 🛠️ Tools & Technologies

* Power BI
* Microsoft Excel / CSV
* Data Visualization Techniques

---

## 📈 Key Insights

* Customers with **month-to-month contracts** have higher churn rates
* **Electronic check** users are more likely to churn
* Customers with **longer tenure** tend to stay longer
* Certain **internet services** show higher churn patterns
